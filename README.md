# Retos públicos de Python I (Campuslands)

## 1. Calculadora de Descuentos
Desarrolla un programa en Python que solicite al usuario el precio de un producto y su categoría. Según la categoría, el programa debe aplicar un descuento y mostrar el precio final.

## 2. Calculadora de Números Primos
Desarrolla un programa en Python que solicite al usuario ingresar un número entero positivo y determine si es un número primo o no.
